# Teaching Notebooks

This repository contains a collection of Jupyter notebooks designed for teaching the “Data Modeling and Visualization” course at HAW Hamburg. Each notebook focuses on a specific topic, illustrating both theoretical foundations and practical implementations using Python and popular data science libraries such as **Pandas**, **NumPy**, **Scikit-learn**, **Seaborn**, and **TensorFlow**.

## Contents

| Notebook | Topic | Description |
|-----------|--------|-------------|
| **02_data_visualization** | Data Visualization | Demonstrates how to create and customize various types of charts using **Pandas** and **Seaborn**. |
| **03_data_cleaning** | Data Cleaning | Covers techniques for handling missing values, detecting and treating outliers, and resolving other common data quality issues. |
| **04_data_transformation** | Data Transformation | Explains how to merge, aggregate, and encode data using **Pandas** operations and **one-hot encoding**. |
| **06_regression** | Regression Analysis | Introduces **linear regression** and model evaluation using **mean squared error (MSE)**. |
| **07_classification** | Classification | Implements **logistic regression** and **k-nearest neighbors (kNN)** classifiers, along with performance evaluation metrics. |
| **08-1_ARIMA** | Time Series Analysis (Part 1) | Presents **ARIMA** modeling for time series forecasting. |
| **08-2_ARIMA** | Time Series Analysis (Part 2) | Demonstrates how to search for optimal **ARIMA** hyperparameters. |
| **09-1_decision_tree_classification** | Decision Trees (Classification) | Builds and evaluates **decision tree classifiers**. |
| **09-2_decision_tree_regression** | Decision Trees (Regression) | Applies **decision tree regression** for continuous target prediction. |
| **11_clustering** | Clustering | Compares clustering techniques including **K-Means**, **Agglomerative Clustering**, and **DBSCAN**; introduces **Local Outlier Factor (LOF)** for anomaly detection. |
| **12_dimensionality_reduction** | Dimensionality Reduction | Explores **PCA** and **t-SNE** for reducing data dimensionality and visualizing complex datasets. |
| **14_deep_learning** | Deep Learning | Implements a simple **feed-forward neural network** for supervised learning tasks. |
| **15_LSTM** | Recurrent Neural Networks | Introduces the **Long Short-Term Memory (LSTM)** model for sequential data analysis. |
| **16_SOM** | Self-Organizing Maps | Demonstrates the use of **Self-Organizing Maps (SOM)** for unsupervised learning and visualization. |

## Usage

Each notebook is self-contained and can be executed independently.
They are organized sequentially to reflect a progressive learning path—from data preprocessing and visualization to advanced machine learning and deep learning models.
